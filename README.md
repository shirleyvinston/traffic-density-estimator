# Traffic Density Estimator

Real-time AI-powered traffic monitoring system using YOLOv8, ByteTrack, OpenCV and Streamlit Dashboard.

## Features

- Vehicle Detection
- Vehicle Tracking
- Traffic Density Estimation
- Real-Time Dashboard
- Traffic Analytics
- Speed Estimation
- Edge AI Ready (Jetson Nano)

## Tech Stack

YOLOv8
OpenCV
ByteTrack
